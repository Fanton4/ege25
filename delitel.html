    Делители

    В данном прототипе 25 задания требуется найти или является ли число простым , или числа у которых определённое колличество делителей

    Рассмотрим пример данного задания

    Напишите программу, которая ищет среди целых чисел, принадлежащих числовому отрезку [185 311; 185 330], числа, имеющие ровно четыре различных натуральных делителя. Для каждого найденного числа запишите эти четыре делителя в четыре соседних столбца на экране с новой строки. Делители в строке должны следовать в порядке возрастания

    Например, в диапазоне [12; 14] ровно четыре различных натуральных делителя имеет число 14, поэтому для этого диапазона вывод на экране должна содержать следующие значения:

    1 2 7 14

    Решим задачу перебором. Будем проверять количество делителей каждого числа из диапазона, если их количество равно четырём  — записываем их в массив d. После этого выводим эти делители на экран в новой строке. 
    for num in range(185311, 185331):
    deliteli = []
    for d in range(1, int(num**0.5) + 1):
        if num % d == 0:
            deliteli.append(d)
            if num//d != d:
                deliteli.append(num//d)
    if len(deliteli) == 4:
        deliteli.sort()
        print(deliteli[0],deliteli[1],deliteli[2],deliteli[3])
    Ответом на данное задание будет являтся список

    1 2 92657 185314

    1 47 3943 185321

    1 241 769 185329
